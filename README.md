# SAE 401


## Languages

Cette SAE fut réalisé avec :

<ul>
  <li>Symfony</li>
  <li>React</li>
  <li>TailwindCSS</li>
</ul>

## Dépendances

### INSTALLER LES DEPENDANCES :

```bash
npm install
```

# Dans le cas où vous souhaitez installer un par un
### INSTALLER LES DEPENDANCES BACK

<ul>
<li>Nelmios Cors Bundle</li>

```bash
composer require nelmio/cors-bundle
```
</ul>

### INSTALLER LES DEPENDANCES FRONT

<ul>
<li>Axios</li>

```bash
npm install axios
```

<li>Chart.js</li>

```bash
npm install chart.js
```

<li>Chartjs-plugin-datalabels</li>
<li>https://www.npmjs.com/package/chartjs-plugin-datalabels</li>

```bash
npm install chartjs-plugin-datalabels 
```

<li>react-router-dom</li>

```bash
npm install react-router-dom
```

</ul>

## Comment lancer le projet

<p> Lancer dans une fenêtre du back </p>

```bash
symfony serve
```

<p> Lancer dans une fenêtre du front </p>

```bash
npm run dev
```
## BDD
